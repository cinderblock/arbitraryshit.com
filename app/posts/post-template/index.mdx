This post is a template (and a living test of the draft + related-post
machinery — it links to the first post, which shows a backlink here in dev).

To start a new post:

1. Copy this folder to `app/posts/<your-slug>/` — the folder name is the URL.
2. Fill in the frontmatter above. `title`, `date` (quoted!), and
   `description` are required. While `draft: true` is set, the post is
   listed in dev but unlisted in production (the URL still works as a
   noindexed preview). Delete the flag to publish.
3. Write. Markdown, code blocks, tables all work.

Images live in the folder and get imported:

```mdx
import photo from "./photo.jpg";

<img src={photo} alt="Describe it" />
```

Interactive components too — any `.tsx` file in the folder:

```mdx
import { Widget } from "./widget";

<Widget />
```

`related:` takes a list of post slugs. Links are symmetric: listing a post
here also puts a backlink on that post, so old posts point forward to new
ones automatically.
