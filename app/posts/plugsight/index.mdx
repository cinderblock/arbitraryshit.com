[Microsoft's Device Manager](https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/device-manager)
has long been a neglected bit of utility software. I've long used tools like
[USB Tree View](https://www.uwe-sieber.de/usbtreeview_e.html) and
[Serial Port Notifier](https://helmpcb.com/software/serial-port-monitor). They're
great and get the job done.

<img
  src={deviceManager}
  alt="Windows Device Manager showing its flat, alphabetical list of device categories"
/>

I was doing a lot of USB development and watching devices disappear and come back,
only to have the whole Device Manager tree repopulate, making it impossible to see
diffs. Claude made the first prototype of a better Device Manager that shows device
deltas clearly and includes all the other handy features of these other device
monitors that I like using. I couldn't help but also include some other helpful
features like filters, parent/child relations, and a full tree view — all the way
down to the PCIe hardware.
